针对 node.js 文件系统的练习，实现常用命令

1.node addfile filename		:创建文件
2.node delfile filename		:删除文件
3.node rename oldname newname	:重命名
4.node cat filename		:查看文件
5.node echo filename data	:写入文件
6.node dir			:列出当前目录下文件
7.node mkdir dirname		:创建目录
8.node rmdir dirname		:删除一个目录
9.node copy src dst		:复制一个文件
